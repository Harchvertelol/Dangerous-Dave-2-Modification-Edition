changelevelongameover=2 // Необязательный параметр, указывающий, как менять уровень при Game Over. Если равен просто числу N, то меняется на уровень с номером N. Если написано "+N", то к текущему номеру добавляется N (с умом,
			// то есть есть %cur_lev% + N > max_lev, то переходим на последний уровень). Если равен "-N", то из текущего уровня вычитается N (с умом, если %cur_lev% - N < 1, то переходим на уровень 1).

numberobjectstilesfield=1 // Необязательный параметр, указывает номер слоя тайлов, после которого рисуются объекты. При отсутствии приравнивается к "numberphysictilelayer". Операции с дверями проводятся на этом уровне.
